# AutoExpertise
ASP.NET MVC and SVG based web application.

Instead of the actual picture(.img, .jpeg etc.) , SVG technology is used for modelling the car.

When the user creates the record of the vehicle, the damage status, dyed / changed and then the color codes will be represented on the vehicle model for changing situations.

Technologies: C#, ASP.NET MVC, SVG, BOOTSTRAP, JAVASCRIPT, MYSQL DATABASE
